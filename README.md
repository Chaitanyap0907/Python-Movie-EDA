“Exploratory data analysis on movie dataset using Python to analyze trends in budget, revenue, and release patterns.”
# Python Movie EDA

## Overview
This project performs exploratory data analysis on a movie dataset to identify trends in budget, revenue, and movie releases.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Data cleaning (removed null values and duplicates)
- Analyzed relationship between budget and revenue
- Studied movie release trends over years

## Key Insights
- Higher budget movies tend to generate higher revenue
- Movie releases have increased over time
- Some high-budget movies result in low returns

## Files
- analysis.ipynb → Python code
- movies.csv → Dataset
- output.png → Visualizations
